# 🎲 RollDice

A simple Java console application that simulates rolling one or more dice.

The program asks the user how many dice they want to roll, validates the input, generates random dice values, displays each die using ASCII art, and calculates the total of all rolls.

---

## Features

- 🎲 Roll one or multiple dice
- ✅ Input validation
- 🎲 Random dice generation
- 🖥️ ASCII art for each dice face
- ➕ Displays the total value of all dice rolled

---

## Technologies Used

- Java
- Scanner
- Random
- Methods
- Loops
- Switch Statements
